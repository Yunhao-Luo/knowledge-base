## Abstract

### Abstract

Seamless integration of physical objects as interactive digital entities remains a challenge for spatial computing. This paper explores Augmented Object Intelligence (AOI) in the context of XR, an interaction paradigm that aims to blur the lines between digital and physical by equipping real-world objects with the ability to interact as if they were digital, where every object has the potential to serve as a portal to digital functionalities. Our approach utilizes real-time object segmentation and classification, combined with the power of Multimodal Large Language Models (MLLMs), to facilitate these interactions without the need for object pre-registration. We implement the AOI concept in the form of XR-Objects, an open-source prototype system that provides a platform for users to engage with their physical environment in contextually relevant ways using object-based context menus. This system enables analog objects to not only convey information but also to initiate digital actions, such as querying for details or executing tasks. Our contributions are threefold: (1) we define the AOI concept and detail its advantages over traditional AI assistants, (2) detail the XR-Objects system’s open-source design and implementation, and (3) show its versatility through various use cases and a user study.

This research explores Augmented Object Intelligence (AOI), a novel interaction paradigm that enables users to engage with physical objects in extended reality environments as if they were interactive digital entities. The study introduces XR-Objects, an open-source prototype system that combines real-time object detection, spatial anchoring, and multimodal large language models to create context-aware interactions with everyday items.

The XR-Objects system leverages several key technologies working in concert. It employs computer vision through MediaPipe for real-time object detection and classification, providing 2D bounding boxes that serve as initial spatial references. The system then uses depth mapping and raycasting to translate these 2D locations into precise 3D world coordinates, enabling accurate spatial anchoring. Each detected object becomes coupled with a multimodal language model that can retrieve detailed semantic information beyond basic object classification, such as product specifications, pricing, and nutritional data. The interface prioritizes an object-centric approach rather than traditional app-centric models, allowing users to directly access digital functionalities by selecting recognized objects without navigating through application menus.

The user interface design emphasizes minimalism and spatial consistency through world-space UI elements anchored to physical objects. Semi-transparent spheres indicate interactable objects, expanding into context menus upon user selection. These menus organize seven implemented actions into four categories: Information retrieval, object comparison, sharing capabilities, and anchoring widgets like spatial timers and notes. The system supports both voice and touch interactions, adapting to user preferences and contextual requirements.

A comparative user study evaluated XR-Objects against a state-of-the-art multimodal language model interface called Chatbot. Eight participants completed tasks in simulated grocery shopping and home scenarios, with results demonstrating significant performance advantages for XR-Objects. Participants using XR-Objects completed all tasks approximately twenty-four percent faster than Chatbot users, with mean completion times of two hundred seventeen seconds compared to two hundred eighty-six seconds. Beyond speed metrics, participants reported higher satisfaction and perceived ease of use with XR-Objects, particularly regarding information retrieval and object comparison tasks.

The research envisions diverse applications spanning cooking, shopping, education, productivity, and smart home control. In cooking scenarios, XR-Objects could provide nutritional information, allergen warnings, and spatially anchored timers. At retail locations, it enables rapid product comparison and allergen filtering. Educational applications could augment physical objects with contextual learning content. The system shows particular promise for headset-based deployment, where participants expressed stronger preferences for XR-Objects over traditional chatbot interfaces, suggesting the paradigm's alignment with emerging wearable AR technologies.

The work addresses fundamental limitations of current AI interfaces by grounding interactions in spatial context, reducing hallucination risks through object-instance-based spatialization, and providing intermediate feedback about system recognition. While acknowledging current technical constraints with commercial headsets regarding camera access, the research provides open-source implementation guidance for future hardware iterations, positioning Augmented Object Intelligence as a pathway toward seamlessly integrated digital and physical world interaction.
